# XI 風サイコロパズル — 仕様書 (spec)

PS1「XI [sai]」風パズル。実装は `src/main.ts` 単一ファイル（Three.js + TypeScript + Vite）。
このドキュメントは、これまでに確定した挙動の仕様をまとめたもの。数値の多くは `CONFIG`（先頭）で調整可能・GUI/Export-Import 対応。

---

## 1. 盤面とサイコロの基本

- 盤面は `GRID×GRID`（既定7×7）。各マスは **スタック（`cells[gx][gz]: Die[]`）** で **最大2段**（`MAX_STACK=2`）。
- 各サイコロは標準ダイス（対面の和7）。`orient`（top/bottom/east/west/south/north）で現在の各面の目を保持。
- 面は**目の数ごとに色分け**（1=赤 / 2=橙 / 3=黄 / 4=緑 / 5=青 / 6=紫、`FACE_COLORS`）。
- サイコロの中心 y = `Y + level*CELL`（`dieLevel` = スタック内 index、0=1段目, 1=2段目）。

### ヘルパー
`height(gx,gz)` / `topDie(gx,gz)` / `isFull` / `pushDie` / `popDie` / `dieLevel` / `dieWorldY(level)`。
`cells` への直アクセスは原則これら経由。

---

## 2. サイコロの状態（ステートマシン）

フラグ: `die.sinking?: {t, level, duration}` / `die.reserved?: boolean` / `die.falling?: {toY, vy}`（いずれも無ければ ACTIVE）。

```
   出現/着地 → ACTIVE ──(同じ段で value個以上連結)──→ SINKING ──(t≥1)──→ 消滅(除去)
                  │  ↑                                   ↑
   直下がSINKING化 │  │ 着地(→再マッチ)                    │ 幽霊の上に落下着地=押しつぶし(即消滅)
                  ↓  │
              RESERVED ──(下段が半分沈む t≥0.5)──→ FALLING ──(着地)──→ ACTIVE
```

### `sinking.t`（沈み進行度 0→1）の意味と2つの境目
- `t = 0`：消え始め（元の高さ）。
- **`t = 0.5`（`sinkPlayable`）= 足場の境目**：ここで `cells` から除去され、当たり判定（乗れる足場）を失う。上に乗っていたサイコロはここから落下開始。
- **`t = 1` = 連結の境目**：完全消滅し `dice`/`scene` から除去。
- `t` は毎フレーム `t += dt / duration` で進む。`duration` は目ごと（後述）。

### 各状態のプロパティ
| 状態 | フラグ | 操作(転がす/押す) | **chain連結対象** | 足場(乗れる) |
|---|---|---|---|---|
| ACTIVE | なし | 最上段なら可 | ○ | ○ |
| SINKING `t<0.5` | sinking | × | ○ | ○ |
| SINKING `0.5≤t<1`（幽霊） | sinking | × | **○** | ×（当たり判定なし） |
| 消滅 `t≥1` | — | — | ×（除去済み） | × |
| RESERVED | reserved | × | × | △(下段に乗って沈む) |
| FALLING | falling | × | × | × |

**重要**: 「足場として乗れるか（t<0.5）」と「chain で連結できるか（t<1）」は**別の境目**。
見えている間（t<1）の沈みかけ（幽霊含む）は連結対象＝チェーンに使える。完全消滅（t≥1）は対象外。

## 3. マッチ／消滅（バニッシュ・リンク・チェーン・ハッピーワン）

### 連結ルール
- **同じ段（レベル）のサイコロ同士**で連結（1段目は1段目、2段目は2段目）。`cells[ax][az][L]`。
- **cells 外でも t<1 の沈みかけ（幽霊）は連結対象**（`ghosts` マップで補う）。`falling`/`reserved` は対象外。
- 連結数（`group.length`）＝ ACTIVE + 見えてる沈みかけ（t<1）。**完全消滅(t≥1)は数に入らない**。

### バニッシュ
- 上面が同じサイコロを、**「目の数」以上**連結すると消える（value≥2）。例: 目3を3個以上。
- 消えるのは `group` 内の **ACTIVE（`!sinking`）** のみ（沈みかけは既に消える）。

### チェーン（連鎖）
- 沈みかけ（t<1、見えてる）に同じ目を隣接させ、連結数が value 以上になると、その新サイコロも消える。
- 例: `4 4 4 4`（沈みかけ4個）に 4(*) を繋ぐ → 5個 → 4(*) も消える。
- 反例: 先行 4×4 が**完全消滅(t≥1)した後**、残った 4(*)（1個）に新4 → 連結2個 < 4 → **消えない**（見えてる4が足りない）。
- 実装: 落下着地・サイコロ移動のたびに `resolveMatches()` を再実行（`while` ループ）。`chain` カウントが増え、スコア倍率に乗る。
- **チェーン戻し（`chainReturn`、既定0.1）**: 2連鎖以上のとき、沈みかけの `t` を少し戻して次のチェーンを繋ぎやすくする。

### リンク（同時消し）
- **1手（1回の判定パス）で複数の独立したグループが同時成立**した数を `link` とする。
- `link` の数だけスコア倍率が上がる。link≥2 は効果音を強調。

### ハッピーワン（目1の特例）
- 目1は単独では消えない。**沈みかけ（toRemove対象）に隣接した最上段の1**があると、盤上の最上段の1を**まとめて全消し**。
- ただし**アクイが乗っている最上段の1は消えない**。

### スコア
`score += 消した数 × 10 × CHAIN倍率 × LINK倍率`。

### 落下と押しつぶし（重力）
- 下段が `t≥0.5` で cells から外れると、上のサイコロが `falling`（重力 `fallGravity`）で落下。
- 落下が着地した結果で `resolveMatches()`（チェーン判定）。
- 落下が**幽霊（沈みアニメ中・cells外）の上に着地**すると、その幽霊を**即消す（押しつぶし）**。

### 消滅時間（目ごと・個別管理）
- `sinkDuration(value) = sinkMs + (value-3) * sinkMsPerPip`（下限500ms）。
- 既定: `sinkMs=5000`（目3）, `sinkMsPerPip=670` → 目6≈7010ms / 目2≈4330ms。**大きい目ほどゆっくり消える**。
- サイコロごとに `sinking.duration` を保持。

## 4. アクイ（プレイヤー）と移動

- 状態: `player = { gx, gz, h, facing, carrying }`。`h` = 立っている段数（0=床/1/2、足元スタック高さから導出）。
- アクイ足元 y = `h*CELL`。乗っているサイコロは**キャラの重みで少し沈む**（`weightSink`、既定0.07。乗り移りで戻る）。
- 向き: **鼻（黄のとがり）**が `facing` を示す。前方の持ち上げ対象は**橙ハイライト**、運搬中の設置先は**水色**ガイド。

### 移動（矢印キー・1マス即移動）
足元 `h` と進行先 `hn = height(隣)` で分岐:
- `hn === h`：**乗り移り/床歩き**（同じ高さの隣へ）。
- `hn === h+1`：**押し**（足元と同じ段＝正面のサイコロを押す。下記）。
- `hn < h`：**転がし**（乗っている最上段を隣へ。空マスなら降りる／1段なら2段目に積む。下段は残る）。
- `hn > h+1`：**登れない**（ジャンプで越える）。

### 押し（回転しない・目は変わらない）
- 正面（足元と同じ段）の最上段サイコロを平行移動。
- **床(h=0)から**: 押し出し先が**空マス(height===0)のときだけ**（＝2段に積まない）。
- **1段の上(h≥1)から（2段目を押す）**: 押し出し先が**2段未満なら可**。低ければ**落ちる**（空→1段目／1段→2段目に乗る）、2段なら押せない。
- 落下する押しは「水平移動 → そのマスで真下に落下」のL字（床抜けを防ぐ）。

### 転がし
- 乗っている最上段を移動方向へ1回転（`orient` も更新＝上面が変わる）。土台（下段）はその場に残る。
- 段差を下る場合は転がりながら落差ぶん下る（`drop`）。

---

## 5. ジャンプ（Z / Shift。XI Jumbo 由来）

- アクイが乗っている**最上段サイコロごと真上に跳ぶ**（素ジャンプ＝床からはサイコロなし）。
- **滞空中に最初に押した矢印**で着地先（ジャンプ元の隣マス）が決まり、**以降の入力は無視**（`jumpLocked`）。
- 着地先が**2段未満**なら、その上に**積む**（隣が1段→2段目／空→1段目）。**2段スタックには積めない・乗れない**（不可なら元へ戻る）。
- 隣へ移った分、**移動方向にサイコロが転がる**（滞空中に回転、上面が変わる）。
- 高さ `hopHeight`、滞空時間 `hopMs`。IME 対策で Z と Shift の両対応。
- 上部プレビュー（後述）はジャンプ中も担いでいるサイコロに連動。

---

## 6. 2段（多段）の仕様

- **最大2段**（3段目不可）。
- **最上段だけ**が操作・マッチ対象。**下段は土台として固定**。
- 積む手段:
  - **ジャンプ**で隣スタックの上に積む。
  - **リフト**で前方スタックの上（2段目）に置く（`placeTarget` は `!isFull` のマス）。
  - **押し**で 2段目を別の1段の上に乗せる（落ちる場合あり）。床からの押しでは積まない。
- 段を**登る**のは原則ジャンプのみ（通常移動・転がしは平ら/下りのみ）。
- 下段が消えると上段は落下（重力）。アクイが乗っていれば一緒に沈み/落ちて1段下・床へ。

---

## 7. リフト（Space）

- 向いている**前方の最上段サイコロ**を持ち上げて頭上に担ぐ（アクイの位置・段は不変）。
- **持ったまま移動・ジャンプ可**。もう一度 Space で**向いた先（空きor1段の上）**へ設置（`!isFull`）。
- 設置後にマッチ判定。沈み中のサイコロは持ち上げ不可。

---

## 8. せり上がり（タイムモード）

- **空マス（height===0）にのみ**新サイコロがせり上がる（`riseMs`、既定1500でゆっくり）。
- 出現間隔 `spawnBase`→経過で `spawnMin` まで短縮（`spawnAccel`）。
- **全マスが1段以上で GAME OVER**（クリックでリスタート）。

---

## 9. 上部プレビュー

- 画面**左上**に「今乗っているサイコロ」を表示（カメラ固定、`miniPivot` を傾けて**3面**が見える角度）。
- 連動対象: 通常=足元サイコロ / **転がし中=転がっているサイコロ** / ジャンプ中=担いでいるサイコロ。乗り移りで切替。

---

## 10. 操作まとめ

| キー | 動作 |
|---|---|
| ← ↑ → ↓ | 移動（1マス。乗り移り/転がし/押し、状況で分岐） |
| Space | リフト（持ち上げ / 設置） |
| Z または Shift | ジャンプ（サイコロごと。空中で方向→隣の上に積む） |

---

## 11. 主な CONFIG（GUI で調整・Export/Import 可）

- 盤面: `grid`, `startDice`
- タイミング: `rollMs`, `moveMs`, `hopMs`, `sinkMs`(目3基準), `sinkMsPerPip`, `sinkPlayable`, `riseMs`
- スポーン: `spawnBase`, `spawnMin`, `spawnAccel`
- 動き/演出: `jumpHeight`(小ジャンプ), `hopHeight`, `liftHeight`, `sinkDepth`, `flashScale`, `particleCount`, `particleSpeed`, `gravity`(粒子), `fallGravity`(落下), `weightSink`, `chainReturn`, `sfxVolume`
- デバッグ: `showLog`（操作/消滅ログを画面右下に表示。「📋 ログをコピー」ボタンあり）

## 12. デバッグログ

画面右下にログ（直近16行表示・全500行コピー可）。`logMsg()` で出力。
操作（歩き/乗り移り/転がし/押し/ジャンプ/持上げ/設置）、消滅（個数・CHAIN・LINK）、落下着地、押しつぶし を記録。
コンソールにも `[XI] …` で出力。
