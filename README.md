# XI 風 サイコロパズル

PlayStation の名作アクションパズル **XI [sai]** のコアメカニクスを、Three.js + TypeScript で簡易再現したものです。

## セットアップ / 起動

```sh
npm install
npm run dev      # http://localhost:5173/ が開発サーバー
```

ビルド（型チェック込み）:

```sh
npm run build    # tsc --noEmit && vite build → dist/
npm run preview  # ビルド結果を確認
```

## 操作（悪魔キャラを動かす）

矢印キー ← ↑ → ↓ でアクイを動かします。サイコロの**上面は 3×3 に分かれていて、まずその上を 1 歩ずつ歩きます**。
**端まで来てさらに同じ方向を押すと、初めて隣のサイコロへ**移ります（＝1 つのサイコロを横断するのに端から端で 3 歩）。
歩いた方向にアクイが向き、その向きが持ち上げ／設置の対象になります。隣へ移るときの挙動（本家どおり）:

- **ダイスの上にいるとき**
  - 進行方向が空マス → 乗っているダイスをその方向へ**転がす**（上面が変わる／悪魔も一緒に移動）
  - 進行方向に別のダイス → その上へ**歩いて乗り移る**
- **床にいるとき**
  - 進行方向のダイスを**押す**（平行移動のみ・**回転しない**）。押した先が空マスのときだけ動く
  - 押せない（先が壁／盤外）ダイスには**よじ登る**
  - 何もなければ床を歩く
- ダイスが消えると、その上に乗っていたアクイは**床に降りる**
- **Space = リフト（XI Jumbo 由来）**
  - **向いている前方（隣）のダイス**を持ち上げて頭上に担ぐ（アクイの位置・状態は変わらない）
  - **持ったまま移動・ジャンプできる**。もう一度 Space で**向いた先の空きマス**へ設置（置けるマスは半透明ガイドで表示）
- **Z または Shift = ジャンプ（XI Jumbo 由来）**
  - その場で**真上に高く跳ぶ**。**滞空中に矢印を押すと、その方向の隣のサイコロの上へ飛び乗る**（空中で行き先を決める）。何も押さなければ元の位置に着地
  - 着地先がダイスなら上に乗り、空マスなら床に降りる。持ったままジャンプも可
  - ※ 日本語 IME がオンだと Z が効かないことがあるため Shift でも跳べる。高さ・滞空時間は GUI の「ホップ高さ」「ジャンプ(Z)」で調整可
- **向きの表示**: アクイの**鼻（黄色のとがり）が向いている方向**を示す。持ち上げ対象になる**前方のダイスは橙色でハイライト**される（持ち運び中は設置先が水色で表示）

## ルール（コア）

- サイコロの **上面の目の数 = 縦横に連結した同じ上面の個数 “以上”** になると消える
  - 例: 上面が「3」のサイコロを 3 個以上つなげると消滅（**バニッシュ**）
  - **チェイン**: 沈みかけのサイコロに同じ目を隣接させると、それも巻き込んで消える（連鎖）
  - **ハッピーワン**: 「1」は単独では消えない。沈みかけに「1」を隣接させると、**アクイが乗っているもの以外**の盤上の 1 がまとめて全消し
- 消えるサイコロは**ゆっくり下へ沈んでいく**。**半分沈むまでは足場として有効**で、その間に隣のサイコロへ乗り移れば**地面に落ちない**（沈み中のサイコロは転がせず、乗り移りで脱出する）。チェインを繋ぐ猶予でもある
  - 「乗れる沈み割合」「消滅(沈み)時間」は GUI で調整可能
- 連鎖（消えた結果さらに条件成立）で**連鎖ボーナス**（CHAIN 倍率）が乗る
- 面は **目の数ごとに色分け**（1=赤 / 2=橙 / 3=黄 / 4=緑 / 5=青 / 6=紫）。色で目を見分けられる

## タイムモード（せり上がり）

- 一定間隔で床から新しいサイコロが**せり上がってくる**（時間経過で間隔が短くなる）
- 盤面が**全マス埋まると GAME OVER**（クリックでリスタート）
- 消すと空きができて延命できる。連鎖で稼ぎつつ盤面を捌くのが目的

## パラメータ調整パネル

画面右上の **XI Params**（lil-gui）パネルで、各種パラメータを実時間で調整できます。

- **Board / Difficulty**: 盤面サイズ・開始ダイス数（変更は自動 Restart で反映）
- **Timing (ms)**: 転がし / 歩き / 消滅(沈み) / せり上がりのアニメ時間
- **Spawn**: せり上がり間隔（開始・最短）と短縮レート
- **Motion / Effects**: ジャンプ高さ / 持ち上げ高さ / 沈み込み深さ / フラッシュ拡大 / パーティクル数・初速 / 重力 / 効果音量
- **⬇ Export (copy JSON)**: 現在の設定を JSON でクリップボード（＆コンソール）に出力
- **⬆ Import (paste JSON)**: JSON を貼り付けて設定を一括反映
- **⟳ Restart**: 盤面リセット

全パラメータは `src/main.ts` 冒頭の `CONFIG` オブジェクトに集約されています。Export した JSON は
そのまま `CONFIG` の初期値としてコードに貼り付けできます（最終組み込み時のチューニング値の固定に便利）。

## 構成

- `src/main.ts` … ゲーム本体（盤面・転がし回転・マッチ判定・キャラ操作・せり上がり・効果音）
- `index.html` / `style.css` … 画面・HUD
- サイコロは全部「標準ダイス（対面の和 7）」。3D 回転（見た目）と論理面（`orient`）を同じ転がしで同期
- 効果音は WebAudio で合成（ライブラリ不要）
