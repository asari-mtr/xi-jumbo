# XI 風 サイコロパズル

PlayStation の名作アクションパズル **XI [sai]** のコアメカニクスを、Three.js + TypeScript で簡易再現したものです。

## セットアップ / 起動

```sh
npm install
npm run dev      # http://localhost:5173/ が開発サーバー
```

ビルド（型チェック込み）:

```sh
npm run build    # tsc --noEmit && vite build → dist/
npm run preview  # ビルド結果を確認
```

## 操作（悪魔キャラを動かす）

矢印キー ← ↑ → ↓ で、アクイを 1 マス単位で動かします。盤面は **最大 2 段** まで積めます（最上段だけが操作・マッチ対象、下段は土台）。
歩いた方向にアクイが向き、その向きが持ち上げ／設置の対象になります。足元とその先の高さで挙動が変わります:

- **サイコロの上にいるとき**
  - 同じ高さの隣 → その上へ**歩いて乗り移る**
  - 1段低い隣／空マス → 乗っている**最上段だけを転がす**（土台は残る。隣の上に積まれる）
  - 1段以上高い隣・2段の上へは通常移動では**登れない**（→ ジャンプで越える）
- **床にいるとき**
  - 前方のスタックの**最上段を押す**（回転しない）。**押し出し先が1段なら2段に積まれる**、空マスなら1段目に。先が2段／壁なら押せない
- **段を登る・2段に積むのはジャンプ**（下記）
- 最上段が消えると下段が露出。アクイは足場が沈むと**1段下／床へ落ちる**
- **Space = リフト**
  - **向いている前方（隣）の最上段ダイス**を持ち上げて頭上に担ぐ（アクイの位置・段は変わらない）
  - **持ったまま移動・ジャンプ可**。もう一度 Space で**向いた先**へ設置（空マスや**1段の上＝2段目**にも置ける。置けるマスは半透明ガイドで表示）
- **Z または Shift = ジャンプ（サイコロごと）**
  - アクイが乗っている**最上段サイコロごと真上に跳ぶ**。**滞空中に矢印を押すと、その方向の隣スタックの上へ着地して積む**（隣が1段→2段目、空→1段目、2段→不可で元に戻る）。何も押さなければ元へ
  - 床からの素ジャンプは隣スタックの上に乗るだけ（積みなし・2段の上には乗れない）
  - ※ 日本語 IME がオンだと Z が効かないことがあるため Shift でも跳べる。高さ・滞空時間は GUI で調整可
- **向きの表示**: アクイの**鼻（黄色のとがり）が向いている方向**を示す。前方の持ち上げ対象は**橙ハイライト**、持ち運び中の設置先は**水色**で表示

## ルール（コア）

- マッチは **同じ段（高さ）のサイコロ同士**で見る（1段目は1段目、2段目は2段目で連結）
- 消えかけのサイコロは「幽霊」: **半分沈むまでは足場（乗れる）が、動かせない**。半分沈むと**当たり判定を失い**、上に乗っていたサイコロが**重力で落下**する
- **チェーン**: 落ちたサイコロが落ちた先で同じ目と揃えば連続して消える。幽霊（消えかけ）の上にサイコロが載ると、幽霊は押しつぶされて即消える
- 消えるのは **目が大きいほど時間がかかる**（基準=目3で約5秒、目が増えるごとに長く）。サイコロごとに個別管理
- サイコロの **上面の目の数 = 縦横に連結した同じ上面の個数 “以上”** になると消える
  - 例: 上面が「3」のサイコロを 3 個以上つなげると消滅（**バニッシュ**）
  - **チェイン（連鎖）**: 沈みかけのサイコロに同じ目を隣接させて連続で消す。HUD の CHAIN が増え、スコア倍率が上がる
  - **リンク（同時消し）**: 1手で複数の独立したグループを同時に消す。HUD の LINK の数だけスコア倍率が上がる（音も強調される）
  - **ハッピーワン**: 「1」は単独では消えない。沈みかけに「1」を隣接させると、**アクイが乗っているもの以外**の盤上の 1 がまとめて全消し
- 消えるサイコロは**ゆっくり下へ沈んでいく**。**半分沈むまでは足場として有効**で、その間に隣のサイコロへ乗り移れば**地面に落ちない**（沈み中のサイコロは転がせず、乗り移りで脱出する）。チェインを繋ぐ猶予でもある
  - 「乗れる沈み割合」「消滅(沈み)時間」は GUI で調整可能
- 連鎖（消えた結果さらに条件成立）で**連鎖ボーナス**（CHAIN 倍率）が乗る
- 面は **目の数ごとに色分け**（1=赤 / 2=橙 / 3=黄 / 4=緑 / 5=青 / 6=紫）。色で目を見分けられる

## タイムモード（せり上がり）

- 一定間隔で床から新しいサイコロが**せり上がってくる**（時間経過で間隔が短くなる）
- 盤面が**全マス埋まると GAME OVER**（クリックでリスタート）
- 消すと空きができて延命できる。連鎖で稼ぎつつ盤面を捌くのが目的

## パラメータ調整パネル

画面右上の **XI Params**（lil-gui）パネルで、各種パラメータを実時間で調整できます。

- **Board / Difficulty**: 盤面サイズ・開始ダイス数（変更は自動 Restart で反映）
- **Timing (ms)**: 転がし / 歩き / 消滅(沈み) / せり上がりのアニメ時間
- **Spawn**: せり上がり間隔（開始・最短）と短縮レート
- **Motion / Effects**: ジャンプ高さ / 持ち上げ高さ / 沈み込み深さ / フラッシュ拡大 / パーティクル数・初速 / 重力 / 効果音量
- **⬇ Export (copy JSON)**: 現在の設定を JSON でクリップボード（＆コンソール）に出力
- **⬆ Import (paste JSON)**: JSON を貼り付けて設定を一括反映
- **⟳ Restart**: 盤面リセット

全パラメータは `src/main.ts` 冒頭の `CONFIG` オブジェクトに集約されています。Export した JSON は
そのまま `CONFIG` の初期値としてコードに貼り付けできます（最終組み込み時のチューニング値の固定に便利）。

## 構成

- `src/main.ts` … ゲーム本体（盤面・転がし回転・マッチ判定・キャラ操作・せり上がり・効果音）
- `index.html` / `style.css` … 画面・HUD
- サイコロは全部「標準ダイス（対面の和 7）」。3D 回転（見た目）と論理面（`orient`）を同じ転がしで同期
- 効果音は WebAudio で合成（ライブラリ不要）
